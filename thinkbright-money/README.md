# ThinkBright Money 💰

**Personal Finance Command Centre** — by ThinkBright

A full personal finance app built for the Jamaican market, targeting young and middle-aged adults who want to take real control of their money.

## Features

- 📋 **Monthly Budget** — Income & expenses with Budget/Actual/Variance tracking. Annual overview with Deficit/Overage.
- 📊 **Net Worth Statement** — Full assets & liabilities tracker with auto-calculated net worth.
- 💳 **Debt Tracker** — Track every debt with priority rating (High/Med/Low), interest rate, maturity date, and monthly payments.
- 📈 **Investment Portfolio** — Track stocks, unit trusts, bonds, CDs and more with % gain/loss.
- 🚗 **Vehicle Cost of Ownership** — 5-year projection of the true cost of owning a vehicle.

## Tech Stack

- React 18
- Vite
- localStorage for data persistence
- No backend required

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deployment

Deployed via [Vercel](https://vercel.com). Auto-deploys on push to `main`.
